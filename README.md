# Github search public repository

This is a simple web app to display public repositories from github based on search term provided.

## System version

* ** ruby 2.7.1**
* ** Rails 5.2.3**

## System dependencies

* rspec-rails
* shoulda_matchers
* bootstrap
* typhoeus
* rails-controller-testing
